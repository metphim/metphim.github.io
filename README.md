# 🎬 Mẹt Phim

**Mẹt Phim** là một dự án web mã nguồn mở (open-source) được xây dựng nhằm giúp mọi người học cách phát triển một website trình chiếu phim đơn giản bằng HTML, CSS và JavaScript.

> “Phim hay cả mẹt” 🍿

---

## 📖 Giới thiệu

Mẹt Phim là một project học tập, được thiết kế để:

* Minh hoạ cách xây dựng một website xem phim cơ bản
* Tách biệt rõ ràng giữa giao diện, logic và dữ liệu
* Giúp người mới dễ đọc, dễ hiểu và dễ chỉnh sửa

Dự án lấy cảm hứng từ hình ảnh **“mẹt”** trong văn hoá Việt Nam — nơi mọi thứ được bày biện gọn gàng và dễ tiếp cận.

---

## ✨ Tính năng

* 🎥 Hiển thị danh sách phim từ file JSON
* 🏠 Trang chủ với phim nổi bật
* ▶️ Xem phim thông qua embed video
* 📂 Cấu trúc project rõ ràng, dễ học
* 📱 Responsive cơ bản
* ⭐ (Có thể mở rộng) Lưu phim yêu thích
